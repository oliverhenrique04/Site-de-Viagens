## 🎯 Objetivo do Projeto

A atividade consistia em replicar com fidelidade um layout proposto (mockup) de uma página de viagens, garantindo a mesma identidade visual e estruturação, além de criar uma segunda página seguindo o mesmo padrão estético.

## ✨ Funcionalidades

- Página Inicial (`index.html`):
  - Hero Section: Banner de destaque com chamada para ação.
  - Destinos Populares: Cards com imagens e descrições curtas dos principais pacotes de viagem.
  - Sobre Nós: Breve descrição da agência.
  - Contato e Localização: Formulário de contato funcional (apenas visual) e informações de endereço com imagem de mapa.
  - Página de Destinos (`destinos.html`):
  - Uma galeria expandida em grid mostrando mais opções de lugares incríveis para explorar, mantendo a coesão visual com a página principal.

## 🛠️ Tecnologias Utilizadas

- HTML5: Semântica estrutural (uso de tags como `<header>`, `<section>`, `<nav>`, `<footer>`, etc.).
- CSS3: Estilização pura, utilizando Flexbox para alinhamento e distribuição dos elementos, gradientes, sombras e responsividade básica.
